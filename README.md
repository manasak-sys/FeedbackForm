# Feedback Form Application

## Description
The Feedback Form Application is a web-based project developed using Django. It allows users to submit feedback through a form, which is stored in the database using Django migrations. The frontend is built using Django templates and styled with CSS for a clean user interface.

## Technologies Used
- Python
- Django
- HTML
- CSS
- SQLite

## Features
- User-friendly feedback form
- Data storage using Django models and migrations
- Form handling using Django templates
- Admin panel to view and manage feedback entries
- Clean and responsive UI using CSS

## Project Structure
- Django MVT architecture
- Models for storing feedback data
- Templates for frontend rendering
- Static files for CSS styling
- Database management using migrations

## How to Run the Project
1. Clone the repository
2. Install Django using `pip install django`
3. Run migrations using `python manage.py migrate`
4. Create superuser using `python manage.py createsuperuser`
5. Start the server using `python manage.py runserver`
6. Open browser and access the feedback form
